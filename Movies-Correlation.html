Movies Correlation With Python

Exploring Movie Correlations using Python for Insights and Recommendations

The goal of this project is Perform an in-depth analysis of a movie dataset to extract valuable insights and understand the relationships between various movie attributes.

Data Loading and Inspection:

  • Utilized Pandas to load the movie dataset from a CSV file.
  • Examined the first few rows of the dataset to understand its structure and content.
  • Checked for missing values in each column.

Data Cleaning:

  • Handled missing values by filling them with appropriate values or dropping them.
  • Converted data types to ensure accurate analysis.

Exploratory Data Analysis (EDA):

  • Explored the distribution of movie ratings, genres, and release years.
  • Conducted a scatter plot to visualize the relationship between movie budgets and gross earnings.

Data Sorting and Duplicates:

  • Sorted the dataset based on gross earnings in descending order.
  • Removed duplicate values based on the 'company' column.

Correlation Analysis:

  • Explored the correlation matrix to identify relationships between numeric features.
  • Created visualizations, including a heatmap, to represent correlations.

Numeric Data Transformation:

  • Converted categorical columns into numeric format for correlation analysis.

Correlation Pairs and Analysis:

  • Analyzed correlation pairs to identify strong relationships.

The project revealed insights into the movie dataset, emphasizing the correlation between movie budget and gross earnings. Key findings highlighted the significance of factors such as votes and release year in influencing a movie's success. The analysis serves as a foundation for deeper investigations and informed decision-making in the movie industry.
