Sentiment Analysis

Analyzing Amazon Reviews for Sentiment Insights Using VADER and ROBERTA Models

The goal of this project is to perform sentiment analysis on Amazon product reviews, aiming to understand the sentiments expressed by customers and gain insights into the overall satisfaction or dissatisfaction with the products.

Tools and Libraries:

  • Python
  • Pandas for data manipulation
  • NumPy for numerical operations
  • Seaborn and Matplotlib for data visualization
  • NLTK (Natural Language Toolkit) for text processing
  • VADER (Valence Aware Dictionary and Sentiment Reasoner) for sentiment scoring
  • Transformers library for utilizing a pre-trained ROBERTA model

Data Loading and Exploration:

  • A bar chart is created to visualize the distribution of review scores.
  • The compound score is visualized across different review scores.

Sentiment Analysis with VADER:

  • VADER sentiment analyzer is applied to the reviews, providing polarity scores for each review.
  • NLTK is used for tokenization and part-of-speech tagging on an example review.

ROBERTA Pretrained Model

  • The project utilizes the ROBERTA pre-trained model from the Transformers library for more advanced sentiment analysis.
  • The ROBERTA model is applied to an example review, producing sentiment scores for negative, neutral, and positive sentiments.

Combined Sentiment Analysis:

  • Sentiment analysis is performed on the entire dataset using both VADER and ROBERTA.
  • The results are combined into a DataFrame for further analysis.

Data Visualization:

  • Pair plots are created to visualize the relationship between sentiment scores and review scores.
  • Examples of reviews with extreme sentiment scores are extracted and displayed.

Sentiment Analysis with Hugging Face's Sentiment Pipeline:

  • The project employs the Hugging Face Sentiment Pipeline using a default model for quick sentiment analysis.

This project provides a comprehensive analysis of sentiment in Amazon reviews, utilizing both rule-based (VADER) and machine learning-based (ROBERTA) approaches. The visualizations and extracted examples offer valuable insights into the sentiments expressed by customers in the dataset.
