Covid-19 Data Explorating in SQL

Exploring COVID-19 Data through SQL Analysis and Visualizing Trends in Tableau

This project involves extracting valuable insights from the COVID-19 dataset using SQL queries and subsequently visualizing key metrics on Tableau. The queries cover diverse aspects, including global and regional infection rates, death percentages, vaccination trends, and population-based analyses. The aim is to provide a comprehensive and interactive visual representation of COVID-19 data for informed decision-making and public awareness.

Key Visualizations:

Global Overview:

  • Tableau visualization showcasing total cases, deaths, and death percentages worldwide.

Total Deaths by Location:

  • Tableau bar chart illustrating the total death count for each location.

Infection Counts and Population Percentage:

  • Tableau scatter plot displaying the highest infection count and percentage of population infected for each location.

Infection Trends Over Time:

  • Tableau time series chart depicting the highest infection count and population percentage over time for each location.
